+ `C4A-network.RData` and `C4A-network.xls` are the same files in different formats, both of which contain Pearson's correlation coefficient (PCC) between *C4A* and every other feature in PsychENCODE
+ Raw and processed PsychENCODE data are available at www.doi.org/10.7303/syn12080241 and [resource.psychencode.org](resource.psychencode.org)
+ Raw count and meta-data for GTEx v7 can be downloaded at the [GTEx Portal](http://www.gtexportal.org/home/datasets)
+ InWeb3 database can be downloaded [here](https://inbio-discover.intomics.com/map.html#downloads)
